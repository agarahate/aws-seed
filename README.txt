# aws-seed
#This is a fully responsive seed for all the devices.
#To run this project follow below steps
open cmd
go to project path and run below cmd
  npm install (this will download all the project dependency)
To start the project run below cmd
  angular-http-server (this will start the project on:localhost:8080)
--------------------------
To add the menu item add the json element in app.contriller
To add the html
  create the html file in views
To add controller
  create js file in scripts/modules and link this in index.html
To add css file
  create css file in stylesheets and link this in index.html
To link html and js
  create the new entry in routes.js
